# Django Wiki Project

A simple wiki web application built using Django.  
Users can search, create, edit, and view encyclopedia entries written in Markdown.

## 🚀 Features

- View encyclopedia entries
- Search for entries (case-insensitive)
- Create new entries
- Edit existing entries
- Random page feature
- Markdown support for formatting

## 🛠️ Technologies Used

- Python
- Django
- HTML
- CSS
- Markdown2

## ⚙️ Installation

1. Clone the repository

git clone https://github.com/yourusername/django-wiki-project.git

2. Navigate to the project directory

cd django-wiki-project

3. Install dependencies

pip install -r requirements.txt

4. Run the server

python manage.py runserver

5. Open in browser

http://127.0.0.1:8000/
